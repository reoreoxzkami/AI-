# かんたん画像編集アプリ

ブラウザで動作するシンプルな画像編集アプリです。

## 主な機能

- 画像アップロード
- 明るさ調整
- コントラスト調整
- ぼかし
- グレースケール
- 左右回転
- 縦横反転
- リセット
- PNG形式で保存

## 使い方

1. `index.html` をブラウザで開く
2. 「画像を選択」で画像をアップロード
3. スライダーとボタンで編集
4. 「画像を保存」でダウンロード

## ローカルサーバーで確認する場合

```bash
python3 -m http.server 8000
```

`http://localhost:8000` にアクセスしてください。
