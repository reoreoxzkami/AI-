# かんたん画像編集アプリ

ブラウザで動作するシンプルな画像編集アプリです。

## 主な機能

- 画像アップロード
- 明るさ調整
- コントラスト調整
- ぼかし
- グレースケール
- 左右回転
- 縦横反転
- リセット
- PNG形式で保存

## 使い方

1. `index.html` をブラウザで開く
2. 「画像を選択」で画像をアップロード
3. スライダーとボタンで編集
4. 「画像を保存」でダウンロード

## ローカルサーバーで確認する場合

```bash
python3 -m http.server 8000
```

`http://localhost:8000/index.html` にアクセスしてください。

## 「Not Found」と表示される主な原因

- **サーバーが起動していない**
  - `python3 -m http.server 8000` を実行してからアクセスしてください。
- **URLが違う**
  - `http://localhost:8000/index.html` を直接開いてください。
- **タスク画面のプレビュー画像URLが期限切れ**
  - `browser:/tmp/...` のようなURLは作業セッション専用で、時間経過後に `Not Found` になる場合があります。
  - その場合は上記のローカルサーバー起動手順で実アプリを確認してください。
